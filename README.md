### Gambling API using Express.js and Heroku DB

## Setup
Use `npm` to install the project dependencies:

```bash
npm install
```

```sh
# Configure your API port
API_PORT=5000

# Auth0 Config
APP_BASE_URL=http://localhost:5000

# MongoDB Config
DATABASE_URL="DB_CONNECTION_STRING"
```

```bash
npm run dev
```
